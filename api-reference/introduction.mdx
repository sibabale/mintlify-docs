<Info>
  The CMOS Account Management API provides standardized, secure access to TCS BaNCS Core Modern Operating System (CMOS) for all account and payment operations.
</Info>

## Overview

The CMOS API simplifies complex mainframe interactions into modern REST endpoints for:
- Premium collections
- Claims payouts
- Investment returns
- Real-time account balance queries
- Automated compliance and audit trails

**Key Features:**
- Real-time transaction processing
- Integration with all major SA banks
- Built-in compliance validation for SARB reporting
- Automatic retry logic for mainframe timeouts
- End-to-end encryption and audit trails

## Authentication

All API endpoints require authentication using either a Bearer token or API key. See the security section in the OpenAPI reference for details.

```json
"security": [
  { "BearerAuth": [] },
  { "ApiKeyAuth": [] }
]
```
