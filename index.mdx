## Welcome to CMOS

The CMOS API provides a unified, secure, and modern interface for all your core banking and payments needs. Built to simplify integration with the TCS BaNCS Core Modern Operating System (CMOS), this API enables:

- Real-time account balance queries
- Seamless debit and credit transactions
- Integration with all major South African banks
- Automated compliance and audit trails
- Robust error handling and retry logic

### Who is this for?
- **Sales Teams:** Automate premium collections and payment capacity checks
- **Claims Departments:** Process payouts and verify balances instantly
- **Private Wealth:** Distribute investment returns and manage client accounts
- **Operations:** Ensure compliance and streamline financial operations
