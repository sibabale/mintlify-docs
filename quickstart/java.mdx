<Info>
  **Prerequisite**: Please install Java (version 8 or higher) and Maven or Gradle before proceeding. <br />
</Info>

Follow these steps to install and run CMOS in your Java application:

**1. Install**

<CodeGroup>

```xml maven
<dependency>
  <groupId>com.cmos</groupId>
  <artifactId>cmos-java</artifactId>
  <version>1.0.0</version>
</dependency>
```

```groovy gradle
dependencies {
  implementation 'com.cmos:cmos-java:1.0.0'
}
```

</CodeGroup>
